1. Примитивные структуры
	•	Массивы
    Последовательный набор элементов фиксированного размера.
	•	Строки
    Последовательности символов (часто рассматриваются как массивы символов).

2. Связанные списки
	•	Односвязный список
    Каждый элемент содержит данные и ссылку на следующий элемент.
	•	Двусвязный список
    Каждый элемент содержит ссылки как на следующий, так и на предыдущий элемент.
	•	Кольцевой список
    Последний элемент ссылается на первый, образуя кольцо.

3. Стек и очередь
	•	Стек (Stack)
    Структура данных по принципу LIFO (последним пришёл — первым ушёл).
	•	Очередь (Queue)
    Структура данных по принципу FIFO (первым пришёл — первым ушёл).
	•	Дек (Deque)
    Двунаправленная очередь, позволяющая добавлять и удалять элементы с обеих сторон.

4. Деревья
	•	Бинарное дерево
    Каждый узел имеет не более двух потомков.
	•	Бинарное дерево поиска (Binary Search Tree, BST)
    Упорядоченное бинарное дерево для быстрой вставки, поиска и удаления.
	•	Сбалансированные деревья
    AVL-деревья, красно-черные деревья – гарантируют логарифмическую глубину.
	•	B-деревья и B+ деревья
    Широко используются в базах данных и файловых системах.
	•	Куча (Heap)
    Специальное дерево, где родительский элемент всегда больше (или меньше) дочерних; применяется для реализации приоритетных очередей.
	•	Фибоначчиева куча (Fibonacci Heap)
    Эффективна в некоторых алгоритмах на графах (например, Дейкстры).
	•	Дерево отрезков (Segment Tree)
    Для эффективного выполнения диапазонных запросов.
	•	Дерево Фенвика (Fenwick Tree, Binary Indexed Tree)
    Для вычисления сумм на отрезках и обновлений за логарифмическое время.
	•	Trie (Префиксное дерево)
    Эффективно для хранения и поиска строк (например, автодополнение).
	•	Суффиксное дерево и суффиксный массив
    Для работы с подстроками в строках.

5. Графы
	•	Представление графов:
	•	Списки смежности
	•	Матрица смежности
	•	Список ребер
	•	Структуры для работы с компонентами связности:
	•	Union-Find (структура непересекающихся множеств)

6. Хэш-таблицы
	•	Хэш-таблица (Hash Table) / Хэш-мапа (Hash Map)
    Обеспечивает быстрый доступ к данным по ключу.

7. Множества и словари
	•	Множества (Set)
	•	Словари (Dictionary) / Ассоциативные массивы

8. Skip List
	•	Вероятностная структура данных, альтернатива сбалансированным деревьям.
